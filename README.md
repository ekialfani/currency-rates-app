## Built With

## Installation
* Clone the repo
  ```sh
  git clone https://github.com/ekialfani/currency-rates-app.git
  ```
* Install dependencies
  ```sh
  npm install
  #or
  yarn install
  ```
* Run the project
  ```sh
  npm start
  #or
  yarn start
  ```
